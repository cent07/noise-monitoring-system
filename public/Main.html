System Overview

The NoiseSense System is an Internet of Things (IoT)-based application developed to monitor and manage environmental noise levels in real time. It is designed for indoor environments such as classrooms and offices where maintaining proper noise levels is important.

The system uses ESP32 devices with sound sensors to capture noise and convert it into decibel (dB) values. These readings are transmitted through WiFi using WebSocket communication for fast and continuous data updates.

A web-based dashboard displays live noise data, graphs, and device information, allowing users to monitor conditions in real time. It also includes alert notifications when noise exceeds set thresholds, as well as reporting features for analyzing noise trends.

Overall, the system provides a simple and efficient way to monitor and control noise using modern IoT technology.

Threshold Guide

🟢 Normal (0–50 dB)

This range represents a quiet and conducive learning environment. Students can focus and understand lessons clearly.

🟡 Moderate (51–70 dB)

Noise is noticeable but still manageable. Prolonged exposure may begin to affect concentration and communication.

🔴 Critical (71 dB and above)

This level may disrupt learning and communication. Immediate attention is recommended to reduce noise.

The threshold levels are based on studies indicating that excessive classroom noise negatively affects students’ concentration, understanding, and overall learning performance. Maintaining noise within acceptable limits helps create a more effective and conducive learning environment.

Administrators may adjust the threshold values depending on room conditions, number of students, and classroom activities. Proper calibration and observation are recommended to ensure accurate and reliable monitoring.

Frequently Asked Questions

❓ What is NoiseSense?

NoiseSense is an IoT-based system that monitors classroom noise levels in real time using ESP32 devices and a web dashboard.

❓ How does the system measure noise?

The system uses a sound sensor connected to an ESP32 device to capture sound and convert it into decibel (dB) values.

❓ What happens when noise exceeds the threshold?

The system triggers alerts and may record audio for verification depending on the configuration.

❓ Why is noise monitoring important in classrooms?

Excessive noise can affect students’ concentration, understanding, and overall learning performance.

❓ Can the system work without internet?

No, the system requires an internet connection to transmit data and display real-time updates on the dashboard.

❓ What does “device offline” mean?

This means the device is not connected to the internet or is powered off.

❓ How can I reconnect a device?

Restart the device and ensure it is connected to the correct WiFi network.

❓ Can I monitor multiple devices?

Yes, the system supports multiple devices and allows users to select specific locations.

❓ What is noise mapping?

Noise mapping shows the noise levels in different locations based on device placement.

❓ Is the audio always recorded?

No, audio is only recorded during critical noise conditions for verification purposes.

❓ Who can access the system?

Only authorized users such as administrators can access the dashboard and features.

❓ Can I adjust the noise threshold?

Yes, administrators can configure the threshold depending on the environment.

❓ What are the reports used for?

Reports help analyze noise patterns over time through daily, weekly, and monthly data.

❓ Is the system accurate?

The system provides reliable estimates, but accuracy depends on proper sensor calibration and placement.

❓ Where should the device be placed?

Devices should be placed in areas where noise monitoring is needed, such as classrooms or study areas.

❓ Can the system be used outside classrooms?

The system is designed for indoor environments, but it can be adapted for other use cases.
